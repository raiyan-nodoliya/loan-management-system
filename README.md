# 🏦 Loan Management System | Spring Boot Project | Java | Thymeleaf | MySQL

A full-stack Loan Management System built using Spring Boot, Thymeleaf, MySQL, and deployed on Railway.

## Live Demo
[Live Project](https://loan-management-system-spring-boot-production.up.railway.app/)

## Features
- Customer loan application submission
- Admin dashboard
- Staff dashboard
- Loan approval and rejection flow
- Risk assessment and recommendation
- Email notifications
- Reports export
- Role-based authentication and authorization

## Tech Stack
- Java
- Spring Boot
- Spring Security
- Thymeleaf
- MySQL
- HTML
- CSS
- JavaScript
- Railway

## How to Run Locally
1. Clone the repository
2. Configure MySQL in application-local.properties
3. Run the project using Maven
4. Open browser at localhost

## Deployment
Deployed on Railway with MySQL database.
